# API Documentation

There are multiple ways to get access and documentation to the server's APIs.

* Swagger.json (Useful for interacting with 3rd party dapps like Postman) - [https://api.phunks.ai/swagger.json](https://api.phunks.ai/swagger.json)
* Swagger URL: [http://api.phunks.ai/api-docs](http://api.phunks.ai/api-docs)
* API MCP (Useful to interacting AI tools): [https://phunksai.readme.io/mcp](https://phunksai.readme.io/mcp)
* Readme.io Documentation: [https://phunksai.readme.io/mcp](https://phunksai.readme.io/mcp)
